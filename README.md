# Notice
<p>Current datapack is for Minecraft 1.21.1; it should support other Minecraft versions if the pack.mcmeta pack format is set correctly:</br>
	1.21.4 = 61</br>
	1.21.5 = 71</br>
	1.21.6 - 8 = 81</br>
	1.21.9 - 10 = 88</br>
	1.21.11 = 90</br>
These versions were not tested nor were older versions. I used <a href="https://misode.github.io/pack-mcmeta/" target="_blank">pack.mcmeta Generator</a> to lookup the supported format integer.
